# QuickPercent

**割合・割引・セット割・フリマ純利益を素早く計算できるWebアプリ**

> 達成率の確認、買い物時の価格比較、フリマ出品の純利益計算など、日常でよく使う「パーセント計算」をブラウザで即座に処理。

---

## 機能一覧

### 割合モード
全体数と成果数を入力して割合（%）を算出。

- `80 / 100 = 80%` のように直感的に入力
- 矢印ボタンで「残り（補数）」表示に切替
- 進捗バー色分け：🔴 0–30% / 🟡 30–70% / 🟢 70–100%
- 履歴保存（無料50件 / Premium 200件）

### セット割モード
「1袋〇〇円、2袋目△△円」のようなセット販売の実質単価を比較。

- 通常単価・2袋目単価を入力 → 実質単価・お得額・割引率を算出
- 比較リストに追加して一覧管理（無料50件 / Premium 100件）

### 割引モード
定価と割引率（または割引後価格）から割引額・売価を計算。

- 割引率とセール価格のどちらかを入力すれば自動補完
- 履歴保存（無料50件 / Premium 200件）

### フリマモード
メルカリ・ラクマ・Yahoo!フリマなどの出品純利益を計算。

- 売値・手数料率・振込手数料・送料・原価を入力
- メルカリ / ラクマ ボタンで手数料を一括設定
- 純利益または入金予定額を表示
- 履歴保存（無料50件 / Premium 200件）

## ローカル開発

```bash
npm install
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開く。

### 環境変数（任意）

`.env.local` を作成：

```bash
cp .env.example .env.local
```

| 変数 | 説明 |
|------|------|
| `NEXT_PUBLIC_SITE_URL` | 本番URL（OGP・sitemap用）例: `https://qp.vercel.app` |

## デプロイ

### Vercel（推奨）

1. [Vercel](https://vercel.com) にリポジトリをインポート
2. 必要な環境変数を設定
3. Deploy

```bash
npm run build   # 本番ビルド確認
npm start
```

---

## データについて

計算履歴はすべて**ブラウザのローカルストレージ**に保存されます。サーバーへは送信されず、プライバシーを保護したまま利用できます。端末・ブラウザを変えると履歴は引き継がれません。
